Detailed Description

Class for Lagrange polynomial basis.

Given a set of data points where no two are the same, the interpolation polynomial in the Lagrange form is a linear combination

of Lagrange basis polynomials

where .
This class implements the one-dimensional function (Lagrange basis) . In this class, the array std::vector<double>& positionValues stores the points and the index is stored in basisIndex.
