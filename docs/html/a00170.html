Detailed Description

template<class V, class M>
class QUESO::VectorSubset< V, M >

A templated class for handling subsets.

This class specifies a subset. The most common example of a subset is a subset of a vector space, present, for instance, in the definition of a scalar function: . is a subset of the set , which is also a vector space.
