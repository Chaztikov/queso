Detailed Description

template<class V, class M>
class QUESO::BaseVectorMdf< V, M >

A templated (base) class for handling MDFs of vector functions.

To obtain the marginal distribution over a subset of multivariate (vector) RVs, one only needs to drop the irrelevant variables (the variables that one wants to marginalize out). If X is a Random Vector which contains the continuous random variables . Then each is a continuous random variable with its own Probability Distribution called the marginal distribution of . This class handles MDFs of a vector RV.
