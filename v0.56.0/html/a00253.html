Detailed Description

template<class V = GslVector, class M = GslMatrix>
class QUESO::WignerVectorRV< V, M >

A class representing a vector RV constructed via Wigner distribution.

This class allows the user to compute the value of a Wigner PDF and to generate realizations (samples) from it.
